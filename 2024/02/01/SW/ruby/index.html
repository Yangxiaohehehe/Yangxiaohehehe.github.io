字符串

1.双引号标记的字符串允许替换和使用反斜线符号，单引号标记的字符串不允许替换，且只允许使用 \ 和 ' 两个反斜线符号。

2.使用序列 #{ expr } 替换任意 Ruby 表达式的值为一个字符串。在这里，expr 可以是任意的 Ruby 表达式
eg：puts “相乘 : #{246060}”;

3.反斜线符号：\n 换行 \r回车 \f换页 \b退格 \e转义 \s 空格 \x 字符x

数组

数组字面量通过[]中以逗号分隔定义，且支持range定义
与其他语言的不通电：

  1. 一个数组类型可以不一样
    eg：ary = [ “fred”, 10, 3.14, “This is a string”, “last element”, ]：

2.通过赋值操作插入、删除、替换元素

3.通过+，－号进行合并和删除元素，且集合做为新集合出现

4.通过<<号向原数据追加元素

5.通过*号重复数组元素

6.通过｜和&符号做并集和交集操作（注意顺序）

哈希

哈希是在大括号内放置一系列键/值对，键和值之间使用逗号和序列 => 分隔。尾部的逗号会被忽略
1.可以通过 new 类方法创建一个空的哈希(可以使用 new 创建带有默认值的哈希)
months = Hash.new “month”

2.可以使用任何的 Ruby 对象作为键或值，甚至可以使用数组
[1,”jan”] => “January”

例子：
months = Hash.new( “month” )
months = {“1” => “January”, “2” => “February”}
keys = months.keys
puts “#{keys}”

范围类型

一个范围表示一个区间
比如（1..5）包含值1,2,3,4,5
（1…5）包含值1,2,3,4

Ruby类

特点：数据封装，数据抽象，多态，继承

例子：
class Customer
@@no_of_customers=0
def initialize(id, name, addr)
@cust_id=id
@cust_name=name
@cust_addr=addr
end
def display_details()
puts “Customer id #@cust_id”
puts “Customer name #@cust_name”
puts “Customer address #@cust_addr”
end
def total_no_of_customers()
@@no_of_customers += 1
puts “Total number of customers: #@@no_of_customers”
end
end

变量：

局部变量：局部变量是在方法中定义的变量。局部变量在方法外是不可用的。在后续的章节中，您将看到有关方法的更多细节。局部变量以小写字母或 _ 开始。

实例变量：实例变量可以跨任何特定的实例或对象中的方法使用。这意味着，实例变量可以从对象到对象的改变。实例变量在变量名之前放置符号（@）。

类变量：类变量可以跨不同的对象使用。类变量属于类，且是类的一个属性。类变量在变量名之前放置符号（@@）。

全局变量：类变量不能跨类使用。如果您想要有一个可以跨类使用的变量，您需要定义全局变量。全局变量总是以美元符号（$）开始。

块

特点：
1.块由大量的代码组成。

2.您需要给块取个名称。

3.块中的代码总是包含在大括号 {} 内。

4.块总是从与其具有相同名称的函数调用。这意味着如果您的块名称为 test，那么您要使用函数 test 来调用这个块。

5.您可以使用 yield 语句来调用块。

格式：
def test
yield 5
puts “在 test 方法内”
yield 100
end
test {|i| puts “你在块 #{i} 内”}

但是如果方法的最后一个参数前带有 &，那么您可以向该方法传递一个块，且这个块可被赋给最后一个参数。如果 * 和 & 同时出现在参数列表中，& 应放在后面。

Begin End

每个 Ruby 源文件可以声明当文件被加载时要运行的代码块（BEGIN 块），以及程序完成执行后要运行的代码块（END 块）。
例子：
BEGIN {

BEGIN 代码块

puts “BEGIN 代码块”
}

END {

END 代码块

puts “END 代码块”
}

MAIN 代码块

puts “MAIN 代码块”

Module

定义：
模块（Module）定义了一个命名空间，相当于一个沙盒，在里边您的方法和常量不会与其他地方的方法常量冲突。

模块类似与类，但有以下不同：

模块不能实例化
模块没有子类
模块只能被另一个模块定义

模块常量命名与类常量命名类似，以大写字母开头。方法定义看起来也相似：模块方法定义与类方法定义类似。

引用：您可以在类方法名称前面放置模块名称和一个点号来调用模块方法，您可以使用模块名称和两个冒号来引用一个常量。

例子:

..

end
end
