workflow model（工作流图）

Def of workflow model（工作流图）：展示整个过程 and 系统支持的过程

流程：Requirement analysis and definition->System and SW design->Implementation(coding)->Testing and Maintenance

箭头代表执行顺序，

Difference 在工作流图和瀑布模型

workflow model 和 Waterfall process Model：

workflow：可以描述任意过程的流程（是一种抽象描述）（普遍描述能描述任意的）

Waterfall process model：是一个软件开发过程（软件开发严格按照顺序执行）（定义的是软件）

Data flow diagram model

Def of Dataflow model：用于展示过程和信息从一个process到另一个的flow

箭头代表的是数据流向

Difference between workflow and DFD

图形上相册，但意义上不同

DFD：箭头更关注于数据流向,箭头上需要描述，描述数据

Workflow：更侧重于活动执行过程，可以不描述箭头

Dynamic modeling

  • State Diagrams
  • Interaction Diagrams

Static Modeling
