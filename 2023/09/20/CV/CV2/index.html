Intro

几何表示：细分曲面，物体的CSG树表示，自然景物表示方法

细分曲线

Chaikin细分

算法步骤：

  1. 每条边的1:3和3:1位置处插入两个点
  2. 把新插入点依次相连
  3. 重复(1)、(2)
拓扑规则:每条边上插入两个点,插入的点按顺序依次相连
几何规则：计算插入点的位置并用图示表示，称之为模板

均匀三次B样条细分

参数形式：C(t)=(−t^3+3t^2−3t+1)p_0+ (3t^3−6t^2+4)p_1+(−3t^3+3t^2+3t+1)p_2+ (t^3)p_3，t∈[0,1]
分段3次多项式:4个控制点确定一段,相邻2段共用3个控制点, 保证C2连续

拓扑规则：每条边上插入一个新顶点，原来顶点保留

几何规则：顶点点/边顶点计算：

v1=1/8v0+6/8v1+1/8v2

vE^1=1/2v0+1/2v1

细分曲面

Def：给定一个粗网格(coarse mesh)，通过加入顶点和面得到更细的网格

方法：三角网格，四边形网格，混合网格

细分模式：几何与拓扑规则

拓扑规则：插入新顶点，边，面

几何规则：计算新旧顶点的新位置

细分模式=初始网格 + 拓扑规则 + 几何规则

Catmull-Clark细分（四边形）

拓扑规则：每条边加入一个点，每个面加入一个点，原来的定点保留

几何规则：

Loop 细分曲面 (1987)（三角网格）

拓扑规则： 每个三角形一分为四.


几何规则：

细分曲面实现

有半边结构：

顶点计算(几何规则)：遍历顶点表，计算顶点新位置，遍历半边表，计算边顶点，遍历面表，生成面顶点(CC细分)

面生成(拓扑规则)：遍历面表: 每个面生成4个面(给出顶点序号)

生成半边结构，重复以上过程




有边表结构：


Vertex list :记录每个顶点的邻接边

Edge list :记录每条边的2个邻接顶点(adjacent vertices)和2个邻接面(adjacent faces)

Face list :每个面的所有顶点和边

细分曲面的优缺点

优点：可以表示任意拓扑，适合动画建模，快速渲染，部分细分曲面有解析表达

缺点:奇异点处没有解析表达，难以计算微分量,奇异点：四边形网格中邻接边数不等于4，三角网格中邻接边数不等于6,不容易构造高阶连续曲面

几何表示

不是光滑的曲面且天天在变的（动态）

比如：山，树，花草,火,云,烟花,流体等

两类方法:

语法规则(L-System based on grammar rules )

粒子系统(Particle system)

分形（Fractal 分形）

特点：

  • 自相似性：局部区域形状与结构和分形本身的形状与结构相似
  • 无限小细节：无论区域多小都有足够的细节
  • 分数维：一般具有分数维


自相似图形的分形维数定义：
S：图形缩放尺度（就比如原来的长度是1，缩放的部分的长度是原来的3分之 1）
n:替换图元的个数
D：维数定义
D = -ln n/ln s n越大，s越接近1，D越大

具体细分方法：就是不断地用新的替代部分迭代

  • 多次迭代
  • 符号的几何解释

粒子系统（Particle system）

组成：粒子

应用：仿真火，雾，烟，流体等

粒子系统的迭代步骤：

  1. c创造一个新粒子并把他们加入系统
  2. 给粒子添加属性（比如初始位置，初始速度，初始大小，初始大小，初始颜色，初始透明度等）
  3. 删除超出生命周期的粒子
  4. 根据粒子的移动属性移动粒子
粒子动力学：力场：重力，风，tension 阻尼：流体，碰撞，弹力
例子死亡：粒子存续时间超出生命周期即删除
粒子绘制：点、球、托球面等
一个简单的粒子系统：
  • 初始化
  • 显示粒子
  • 更新位置
  • 碰撞检测
